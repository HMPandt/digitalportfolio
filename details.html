Wat heb je er van geleerd?

Tijdens de ontwikkeling van mijn portfolio, ik heb geleerd zoveel geleerd over wat je moet doen voor je begint met de coderen, over hoe je moet een website coderen en mooi maken via css. Aan het begin had ik echt geen kennis met programmeren, maar nu heb ik wel de basis om een mooie website te maken. Ik weet ook nu hoe veel coderen je moet doen, ook had ik gespeld met css en ik heb veel geleerd over de mogelijkheden van het, en waarom het zo belangrijk is.

Ik heb ook geleerd hoeveel aandacht wordt besteed aan het beslissen van welke kleuren te gebruiken. Ik had ook nooit gedacht dat kleuren zo’n effect zouden hebben op de psychologie van de kijker.

Wat zou je de volgende keer anders doen?

Volgende keer zou ik wel animaties toevoegen en een beter design maken, ik wil live backgrounds en slideshows gebruiken omdat het gebruik minder ruimte een het is wel mooi. Ik wil ook meer werken met css om het mooier te maken

Wat zou je nog willen leren?

Ik wil leren hoe je animaties gebruiken en coderen, ook wil ik meer weten over javascript. Ik wil ook leren hoe je meer efficiënt coderen en een mooie website maken. En ik heb gehoord dat het is makkelijker om een website te maken met flex box een grids zo ik zou dat ook willen leren.
